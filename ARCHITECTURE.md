# 🌍 Wandr - Smart Travel Planner
## Architecture: Template-Based (Non-AI) with Real Data Enrichment

### 📊 **What Changed**

| Feature | Before (LLM-based) | After (Template-based) |
|---------|-------------------|----------------------|
| **Core Engine** | Groq LLaMA 3.3 AI | Rule-based template matching |
| **Data Source** | Generated by LLM | JSON templates + Real APIs |
| **Cost** | Pay per API call | Free (no LLM costs) |
| **Speed** | ~2-5 seconds | ~0.5-1 second |
| **Transparency** | Black box AI | Clear logic |
| **Dependencies** | groq, pydantic | requests |

---

### 🏗️ **New Architecture**

```
User Input (destination, days, budget, food, interests)
    ↓
Validation Layer
    ↓
Itinerary Engine (Rule-based)
    ├─ Load Template from JSON
    ├─ Enrich with Real Data (APIs)
    │  ├─ Weather (open-meteo.com)
    │  ├─ Destination Info (Wikipedia)
    │  └─ Distance Calculation (haversine)
    ├─ Filter Attractions by Interests
    ├─ Filter Restaurants by Food Preference
    ├─ Build Schedule (9 AM - 8 PM)
    └─ Calculate Budget Breakdown
    ↓
Formatted Itinerary (JSON)
    ↓
Save to Database + Display to User
```

---

### 📁 **New Files Created**

1. **`data/destinations.json`** - Destination templates with:
   - Attractions (entry fee, opening hours, rating)
   - Restaurants (cuisine, price, rating)
   - Hotels (price per night)
   - Local transport info

2. **`utils/travel_api.py`** - Real data fetcher:
   - `TravelDataAPI` class
   - Free APIs: open-meteo, Wikipedia, haversine
   - Caching & error handling

3. **`utils/itinerary_engine.py`** - Non-AI itinerary builder:
   - `ItineraryEngine` class
   - Template filtering by interests
   - Smart schedule building
   - Budget calculation
   - Practical tips generation

### 🚀 **How to Add More Destinations**

Edit `data/destinations.json` and add:

```json
"Barcelona": {
  "country": "Spain",
  "coordinates": {"lat": 41.3851, "lng": 2.1734},
  "attractions": [
    {"name": "Sagrada Familia", "type": "monument", "entry_fee_inr": 2500, ...}
  ],
  "restaurants": [...],
  "hotels": [...],
  "transport": [...]
}
```
